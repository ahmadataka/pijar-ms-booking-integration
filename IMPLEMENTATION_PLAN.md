# Week 1 Implementation Plan

## Objective

Build a backend that can ingest booking information from Microsoft first, without Odoo.

## Step-by-step

1. Create project skeleton
2. Add configuration loading
3. Add Microsoft Graph token acquisition
4. Add room fetch endpoint
5. Add booking fetch endpoint
6. Add normalization layer
7. Add local inspection endpoints
8. Validate with existing pilot rooms

## Deliverable by End of Week

A backend service that can:
- start locally
- authenticate to Microsoft Graph
- fetch room information
- fetch selected booking information
- normalize it into a predictable internal structure
